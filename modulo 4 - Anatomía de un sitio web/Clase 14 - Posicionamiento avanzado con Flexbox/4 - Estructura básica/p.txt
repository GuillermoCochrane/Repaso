Disponiendo UL's


Veamos el alcance que tiene Flexbox. 

Debemos convertir al <ul> con clase "nav-bar" en un contenedor flex para que sus elementos se dispongan uno al lado del otro.
Dejamos una imagen de cómo se ve el sitio de momento:


-----------------------------------------------------------------------------------------------------------------------------------------------------

<html>
  <head>
    <title>Flexbox</title>
  </head>
  <body>
    <ul class="nav-bar">
          <li><a href="">Home |</a></li>
          <li><a href="">Contacto |</a></li>
          <li><a href="">Preguntas frecuentes |</a></li>
          <li><a href="">Escribinos |</a></li>
      </ul>
  </body>
</html>

-----------------------------------------------------------------------------------------------------------------------------------------------------

.nav-bar {
  display: flex; 
}

Respetando medidas en el contenedor
En este ejercicio contamos con una estructura de seis cajas definidas en el index.html y algunas reglas de CSS declaradas por detrás.

Queremos que el <div> con clase "contenedor" se convierta en un flex-container. Pero atención: cada una de las cajas ya tiene un ancho definido y queremos que el contenedor respete ese ancho.

Deberemos asignarle las propiedades necesarias a la clase "contenedor" para poder lograr ese objetivo.

Dejamos una imagen de cómo se ve el sitio de momento:

-----------------------------------------------------------------------------------------------------------------------------------------------------

<html>
  <head>
    <title>Flexbox</title>
  </head>
  <body>
    <div class="contenedor">
        <div class="caja-1">Caja 1</div>
        <div class="caja-2">Caja 2</div>
        <div class="caja-3">Caja 3</div>
        <div class="caja-4">Caja 4</div>
        <div class="caja-5">Caja 5</div>
        <div class="caja-6">Caja 6</div>
    </div>
  </body>
</html>

-----------------------------------------------------------------------------------------------------------------------------------------------------

.contenedor {
  display: flex;
  flex-wrap: wrap;
}
