4 - Formularios avanzados

Seleccionando el mes

Vamos a crear un pequeño formulario con un desplegable para que el usuario pueda seleccionar el mes de su cumpleaños.

Para eso, deberemos crear una etiqueta < select > con atributo y valor name="mes". Dentro del < select >, tendremos que crear 12 etiquetas < option >, una por cada mes, aclarando en cada caso el texto que leerá el usuario: Enero, Febrero, Marzo, etc. No olvidemos, además, escribir el atibuto value en cada una de las etiquetas < option >. Pero ¡atención!, queremos que, como valor, ese atributo tenga el número del mes al que está representando. Por ejemplo: 1 para Enero, 2 para Febrero, y así...

Atención: Para poder averiguar, luego, qué es lo que eligió el usuario, no debemos olvidarnos de agregarle el atributo value a cada < input >.

Escribir comentario

Queremos saber qué opinan nuestros usuarios.

Para eso, debemos agregar, en el formulario que ya está declarado, la etiqueta que le permite al usuario escribir un comentario. Además, deberemos crear dos etiquetas < button>, configurando el atributo type según corresponda, para que el usuario pueda ver un botón que diga "Enviar" y otro que diga "Borrar", que al hacerle click borre lo que ya escribió.
