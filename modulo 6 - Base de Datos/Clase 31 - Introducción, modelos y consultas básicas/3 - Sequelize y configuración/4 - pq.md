2. Dentro del patrón de diseño MVC, ¿qué representa un modelo?

    - Representa la interfaz con la que interactúa el usuario.

    - Contiene una representación de los datos que maneja el sistema, su lógica de negocio, y sus mecanismos de persistencia. **(X)**

    - Actúa como intermediario entre los datos y la vista.


¡Perfecto! Un modelo es la representación de una tabla.
